What is Haircomb?
--------------------

Haircomb is an anonymous, quantum-proof cryptocurrency that is hidden within the transactions of the Bitcoin Core blockchain. Haircomb was created by someone using the name Natasha Otomoski, who published a `whitepaper`_ about Haircomb on October 25, 2019.


.. _whitepaper: https://raw.githubusercontent.com/natasha-otomoski/haircomb/master/WhyTheCombOfNatashaOtomoskiHas21Teeth.txt
